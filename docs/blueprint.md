# **App Name**: StyleAI Advisor

## Core Features:

- Image Input: Upload an image of the user, specify occasion, and gender for style analysis.
- Location Access: Access current location for weather information automatically.
- Style Analysis: Analyze image for skin tone and clothing colors and recommend outfits.
- AI Recommendation: Leverage Gemini API as a tool, providing skin tone, clothing colors, occasion, gender, and weather to give accurate recommendations.
- Feedback Display: Provide feedback to the user.
- Recommendation Output: Display feedback and clothing suggestions in a clear format.

## Style Guidelines:

- Primary color: Soft lavender (#E6E6FA) to convey elegance and approachability.
- Background color: Light gray (#F5F5F5) for a clean and modern feel.
- Accent color: Muted plum (#9370DB) for highlights and calls to action.
- Body and headline font: 'Inter' for a modern and neutral aesthetic.
- Use clean, minimalist icons to represent different clothing categories and weather conditions.
- Maintain a clean and intuitive layout with a focus on visual clarity.
- Use subtle transitions and animations to enhance user experience.
